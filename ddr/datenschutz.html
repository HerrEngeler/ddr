2. Allgemeines zur Datenverarbeitung

Diese Website ist ein nicht-kommerzielles Bildungsprojekt. Es werden keine personenbezogenen Daten erhoben, gespeichert oder weitergegeben. Es gibt kein Backend, keine Datenbank und keinen Server, der Nutzerdaten empfängt.

3. Lokaler Speicher (localStorage / sessionStorage)

Zur Speicherung des Spielfortschritts wird die Web-Storage-API Ihres Browsers verwendet:

  • localStorage (Schlüssel: ddr_progress_v1): Speichert, welche Aufgaben und Akten abgeschlossen wurden. Diese Daten verlassen niemals Ihr Gerät und werden nur lokal im Browser gespeichert.
  • sessionStorage (Schlüssel: ddr_teacher_v1, ddr_music_v1, ddr_track_v1): Temporäre Sitzungsdaten für den Lehrerbereich und den Musikplayer. Werden beim Schließen des Browser-Tabs gelöscht.

Sie können alle gespeicherten Daten jederzeit über die Entwicklertools Ihres Browsers (F12 → Application → Storage) einsehen und löschen.

4. Google Fonts

Diese Website verwendet Google Fonts (Playfair Display, Lato, Courier Prime). Beim Laden der Schriften wird eine Verbindung zu Googles Servern hergestellt, wobei Ihre IP-Adresse übermittelt werden kann. Weitere Informationen: policies.google.com/privacy

Für den Unterrichtseinsatz ohne Google Fonts empfiehlt sich, die Schriften lokal einzubinden oder den Link-Tag im <head> jeder HTML-Datei zu entfernen (die Website funktioniert auch mit System-Schriften).

5. Hosting (GitHub Pages)

Sofern diese Website über GitHub Pages gehostet wird, gelten die Datenschutzrichtlinien von GitHub. GitHub kann Server-Logs erstellen, die IP-Adressen der Besucher enthalten.

6. Keine aktive Datenerhebung

Es gibt keine Analyse-Tools (Google Analytics, Matomo o.ä.), keine Cookies von Drittanbietern, keine Werbung und keine Newsletter.

7. Rechte der betroffenen Personen

Da keine personenbezogenen Daten erhoben werden, entfallen Auskunfts-, Berichtigungs- und Löschungsansprüche gemäß DSGVO in Bezug auf diese Website. Gespeicherte localStorage-Daten können Sie selbst jederzeit über Ihren Browser löschen.
